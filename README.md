User Manual
How to Use the Application:
Login/Register: Create an account or log in using credentials.
Dashboard: View available doctors and appointment slots.
Book Appointment: Select a doctor, date, and time, then confirm booking.
Manage Appointments: View, reschedule, or cancel appointments.
Notifications: Receive confirmation and reminders via email.
